인터넷 인프라의 핵심 요소들

💻 사용자의 노트북

수업에서 사용하는 노트북처럼 인터넷에 연결된 장치입니다. 웹사이트에 정보를 요청하고 결과를 화면으로 받습니다.

🏠 라우터 (Wi-Fi)

집의 모든 기기를 인터넷에 연결해주는 장치입니다. 무선(WiFi) 또는 유선(LAN)으로 연결됩니다.

🌐 ISP (인터넷 서비스 제공자)

KT, SKB, LG U+ 같은 인터넷 회사입니다. 집의 라우터를 인터넷 전체에 연결해줍니다.

🔍 DNS (도메인 네임 서버)

"youtube.com"이라는 주소를 실제 서버의 IP 주소로 변환해줍니다. 인터넷의 "전화번호부" 같은 역할입니다.

☁️ 웹 서버

유튜브, 구글 같은 서비스들이 운영하는 거대한 컴퓨터들입니다. 24시간 우리의 요청에 응답합니다.

📦 데이터 센터

수백만 대의 서버가 모여있는 거대한 건물입니다. 영상, 사진, 정보 등을 저장하고 제공합니다.

💡 재미있는 사실: 유튜브의 동영상 데이터는 전 세계 여러 데이터 센터에 복사되어 저장됩니다. 사용자는 보통 가까운 서버에서 영상을 받아 더 빠르게 재생할 수 있습니다!

📺 유튜브로 배우는 인터넷 사용

유튜브는 인터넷 인프라가 얼마나 잘 작동하는지 보여주는 좋은 예입니다. 영상은 매우 크기 때문에 빠르고 안정적인 연결이 필수입니다.

🎬 동영상 스트리밍

유튜브는 영상 전체를 한 번에 다운로드하지 않고, 조금씩 나누어서 보냅니다. 이를 "스트리밍"이라고 합니다.

🔄 CDN (콘텐츠 전송 네트워크)

전 세계 여러 위치의 서버에 같은 영상을 저장합니다. 사용자와 가까운 서버에서 빠르게 받을 수 있습니다.

📶 적응형 품질 조정

인터넷 속도가 느리면 자동으로 화질을 낮춥니다. 이렇게 해야 영상이 끊기지 않고 재생됩니다.

인터넷 연결의 과정

1단계: 사용자가 "youtube.com"을 입력합니다.

2단계: 사용자의 기기가 DNS 서버에 "youtube.com의 실제 주소는 뭐예요?"라고 물어봅니다.

3단계: DNS가 "유튜브 서버의 IP 주소는 142.251.41.14입니다"라고 답합니다.

4단계: 사용자의 기기가 그 서버에 요청을 보냅니다.

5단계: 유튜브 서버가 웹 페이지와 영상을 사용자에게 보냅니다.

6단계: 웹 브라우저가 받은 정보를 화면에 표시합니다.

⚙️ 6단계 연결 과정

1주소 입력youtube.com
2DNS 질문실제 IP 찾기
3IP 응답142.251...
4서버 요청영상 페이지 요청
5데이터 전송HTML, 이미지, 영상
6화면 표시브라우저 렌더링

이 여섯 단계가 대부분 1~2초 안에 이어져서 하나의 웹페이지처럼 보입니다.

🤖 Class 2: ChatGPT 작동 원리

우리는 이미 AI를 매일 사용하고 있습니다!

AI는 더 이상 먼 미래의 기술이 아닙니다. 우리는 이미 일상에서 AI를 활용하고 있습니다.

💬 ChatGPT - 생성형 AI의 작동 원리

ChatGPT는 공개 데이터, 라이선스 데이터, 사람이 만든 예시 등 다양한 텍스트를 학습한 AI입니다. 사용자가 질문하면 학습한 언어 패턴을 활용해 적절한 답변을 만들어냅니다.

💬 ChatGPT 대화 과정

1
사용자 질문

사용자가 궁금한 내용을 자연어로 입력합니다.

"AI가 뭐야?"
2
문맥과 패턴 분석

질문의 의도, 앞뒤 문맥, 학습된 언어 패턴을 함께 참고합니다.

핵심 단어 + 대화 흐름 + 다음 단어 예측
3
답변 생성

가장 자연스러운 문장을 이어 붙이며 사용자에게 답변을 보여줍니다.

"AI는 사람이 만든 지능형 프로그램입니다."
❓ 질문 이해

작동 방식: AI가 질문에서 핵심 단어와 의도를 파악합니다.

예시: "AI가 뭐야?" → 'AI', '정의', '설명' 같은 핵심 요소 인식

🔍 문맥 파악

작동 방식: AI가 전체 대화 흐름을 참고해 적절한 수준의 답변을 준비합니다.

예시: 같은 질문이어도 어린이에게는 쉽게, 전문가에게는 깊이 있게 답변

📝 답변 생성

작동 방식: 학습한 패턴을 바탕으로 자연스러운 문장을 단어 하나씩 만들어냅니다.

예시: "인공" → "인공지능" → "인공지능의" → "인공지능의 약자는..."

📱 스마트폰에서의 AI

🔍 얼굴 인식 (Face Recognition)

어디서?: 잠금 해제, SNS 사진 태깅

원리: AI가 등록된 얼굴의 특징을 비교해 같은 사람인지 판단합니다.

🎤 음성 인식 (Voice Recognition)

어디서?: "Siri", "Google Assistant", "Alexa"

원리: AI가 사용자의 음성을 텍스트로 변환하고 명령의 의미를 파악합니다.

📸 사진 최적화

어디서?: 스마트폰 카메라의 야간 모드, 필터

원리: AI가 사진의 밝기, 색감, 선명도를 자동으로 조정합니다.

🔤 생성형 AI (최신 기술!)

💬 텍스트 생성 AI 작동 원리

❓ 질문 입력 "AI란 뭐예요?"
🔤 단어 조각 분석
AI란뭐예요
🧠 다음 표현 예측
✅ 답변 생성 "AI는 인공지능의 약자로..."
💬 ChatGPT / Gemini / Claude

역할: 질문에 답하고, 글을 쓰고, 코드를 작성합니다.

원리: 많은 단어와 문장 패턴을 학습해 다음에 올 표현을 예측합니다.

예시: "AI란 뭐야?" → 자연스러운 문장으로 답변

🎨 이미지 생성 AI (DALL-E, Midjourney)

🐍 Class 3: VSCode 주피터 노트북으로 계산기 만들기

오늘의 목표

처음부터 어려운 챗봇 코드를 만들기보다, 먼저 VSCode에서 주피터 노트북을 준비하고 파이썬으로 숫자를 계산해보겠습니다.

노트북 셀에서 덧셈, 뺄셈, 곱셈, 나눗셈을 바로 실행해본 뒤, 마지막에는 간단한 계산기 프로그램을 만들어 봅니다.

🧰 Miniconda

파이썬 실행 환경을 관리하는 도구입니다. 수업마다 필요한 패키지와 파이썬 버전을 깔끔하게 분리할 수 있습니다.

📝 VSCode

코드를 작성하고 실행하는 편집기입니다. Jupyter 확장을 설치하면 노트북 셀 단위로 코드를 실행할 수 있습니다.

📓 Jupyter Notebook

코드와 실행 결과, 설명을 한 화면에 남길 수 있는 수업용 작업장입니다. 초보자가 실험하며 배우기 좋습니다.

1단계: Miniconda 설치

Windows에서는 Miniconda를 설치한 뒤, 시작 메뉴에서 Anaconda Prompt를 열어 파이썬 환경을 준비합니다.

설치 파일은 Miniconda 공식 다운로드 목록에서 확인할 수 있습니다.

설치 확인 명령어

목표: 파이썬과 conda가 제대로 설치되었는지 확인합니다.

Anaconda Prompt에서 실행: python --version

추가 확인: conda --version

2단계: 수업용 가상환경과 주피터 설치

가상환경은 프로젝트별 작업 공간입니다. 여기에 Jupyter를 설치하면 VSCode에서 노트북을 실행할 수 있습니다.

환경 생성과 활성화

새 환경 만들기: conda create -n class python=3.11

환경 켜기: conda activate class

주피터 설치: conda install notebook ipykernel

화면 앞에 (class)가 보이면 성공입니다.

3단계: VSCode에서 주피터 노트북 열기

VSCode에서 Python 확장과 Jupyter 확장을 설치한 뒤, 수업 폴더 안에 노트북 파일을 만듭니다.

노트북 준비

폴더 만들기: 예를 들어 python_basics 폴더를 만듭니다.

확장 설치: VSCode Extensions에서 Python과 Jupyter를 설치합니다.

파일 만들기: python_basics.ipynb 파일을 만들고 커널을 class 환경으로 선택합니다.

👋 첫 노트북 셀 실행

# 첫 번째 셀
print("안녕하세요!")
print("파이썬 수업을 시작합니다.")

셀 왼쪽의 실행 버튼을 누르거나 Shift + Enter로 실행합니다.

4단계: 덧셈과 뺄셈부터 시작하기

코딩은 컴퓨터에게 순서대로 일을 시키는 것입니다. 주피터 노트북에서는 셀 하나씩 실행하면서 결과를 바로 확인할 수 있습니다.

➕ 계산 연습

# 계산 연습 셀
print(10 + 3)
print(10 - 3)
print(10 * 3)
print(10 / 3)

a = 10
b = 3
print(a + b)
print(a - b)

숫자를 직접 바꾸면 결과도 바뀝니다. 이것이 변수의 핵심입니다.

5단계: 사용자 입력 받기

input()은 사용자가 키보드로 입력한 값을 받아옵니다. 노트북 셀에서도 입력창이 나타나며, 계산기를 만들려면 숫자와 연산자를 입력받아야 합니다.

⌨️ 입력값으로 계산하기

# 입력값 계산 셀
num1 = int(input("첫 번째 숫자: "))
num2 = int(input("두 번째 숫자: "))

print("더하기:", num1 + num2)
print("빼기:", num1 - num2)

input()은 글자로 입력되기 때문에, 숫자 계산을 하려면 int()로 숫자로 바꿔야 합니다.

6단계: 연산자 입력받기

계산기를 만들려면 숫자뿐 아니라 사용자가 어떤 계산을 원하는지도 입력받아야 합니다. +, -, *, / 같은 기호를 글자로 입력받고, 조건문으로 어떤 계산을 할지 정합니다.

➕ 연산자 선택 연습

# 연산자 입력 연습 셀
op = input("연산자를 입력하세요 (+, -, *, /): ")

if op == "+":
print("더하기를 선택했습니다.")
elif op == "-":
print("빼기를 선택했습니다.")
elif op == "*":
print("곱하기를 선택했습니다.")
elif op == "/":
print("나누기를 선택했습니다.")
else:
print("+, -, *, / 중 하나를 입력해야 합니다.")

input()으로 받은 값은 글자입니다. 그래서 op == "+"처럼 따옴표 안의 글자와 비교합니다.

💡 연산자 기호 기억하기:
  • • 더하기: +
  • • 빼기: -
  • • 곱하기: * 예: 3 * 4
  • • 나누기: / 예: 10 / 2

7단계: 간단한 계산기 앱 만들기

이제 숫자 입력, 연산자 입력, 조건문을 합쳐서 작은 계산기 프로그램을 완성합니다.

🧮 계산기 앱 예제

# 계산기 셀
num1 = float(input("첫 번째 숫자: "))
op = input("연산자 선택 (+, -, *, /): ")
num2 = float(input("두 번째 숫자: "))

if op == "+":
result = num1 + num2
elif op == "-":
result = num1 - num2
elif op == "*":
result = num1 * num2
elif op == "/":
result = num1 / num2
else:
result = "지원하지 않는 연산자입니다."

print("결과:", result)

셀을 실행한 뒤 첫 번째 숫자, 연산자, 두 번째 숫자를 차례대로 입력합니다.

🚀 다음 단계: 계산기까지 만들면 코딩의 기본 흐름을 이해한 것입니다:
  • • 코드는 위에서 아래로 실행됩니다
  • • 변수는 값을 저장하는 이름표입니다
  • • 입력값을 받아 프로그램이 다르게 행동할 수 있습니다
  • • 조건문을 사용하면 상황에 맞게 다른 계산을 할 수 있습니다

Q1: 왜 유튜브는 사용자가 본 영상을 알고 추천할 수 있을까요?
A: AI가 사용자의 시청 데이터를 분석해서 관심사를 파악하기 때문입니다.
Q2: 스마트폰이 사용자의 얼굴을 인식할 수 있는 이유는?
A: 처음 설정할 때 여러 각도에서 얼굴 정보를 등록하고, AI가 그 특징을 비교하기 때문입니다.
Q3: ChatGPT가 자연스러운 한국어로 답변할 수 있는 이유는?
A: 다양한 한국어 텍스트와 대화 예시를 학습했기 때문에 한국어 문장 패턴을 알고 있습니다.
Q4: Ollama는 왜 ChatGPT와 다를까요?
A: Ollama는 내 컴퓨터에서 직접 실행되며, 모델을 미리 내려받아 두면 인터넷 연결 없이도 작동하고, 별도 API 비용이 들지 않습니다.
Q5: 바이브코딩으로 챗봇 웹사이트를 만들 때 중요한 과정은?
A: 만들고 싶은 기능을 설명 → AI가 코드 제안 → 실행 결과 확인 → 오류와 디자인을 다시 수정
